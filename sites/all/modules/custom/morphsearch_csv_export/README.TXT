-- SUMMARY --
Creates a link to export Apache Solr search results as CSV and adds an export link to the search results page.


-- INSTALLATION --

1. Install the morphsearch_csv_export Drupal module as you would any Drupal module.
2. Enable the morphsearch_csv_export module in the administration tools (admin/modules).
3. Configure the module (admin/config/morphsearch_csv_export)

-- CONFIGURATION --
To use the module in a live environment, the network accessible Apache Solr URL has to be set in the configuration menu
(admin/config/morphsearch_csv_export). During the installation an URL for local development will be set automatically.
